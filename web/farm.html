Setup Guide

Gmail App Password (for IMAP)

  1. Enable 2-Step Verification: Google Account → Security → 2-Step Verification → Turn on
  2. Generate App Password: Google Account → Security → App passwords
  3. Select "Mail" as app, "Other" as device (name it "Farm")
  4. Copy the 16-char password (format: abcd efgh ijkl mnop) — paste into IMAP Password field above
  5. Note: This is NOT your Gmail login password. App Passwords bypass 2FA for IMAP access.

Cloudflare Catch-all Email Domain

  1. Go to Cloudflare Dashboard → select your domain → Email → Email Routing
  2. Enable Email Routing (follow DNS setup prompts — Cloudflare adds MX records automatically)
  3. Go to Routing Rules → Catch-all address → Edit
  4. Set action: "Send to destination" → enter your Gmail address
  5. Save. Now any email sent to *@yourdomain.com forwards to your Gmail.
  6. Use this domain in the "Email Domain" field above (e.g. yourdomain.com)
  7. The farm generates random addresses like abc123@yourdomain.com — all forward to your Gmail, and farm.py reads OTPs via IMAP.
